# cms-upload-poc

## Build Instructions

_The following is done using your pc terminal console and NodeJS pre-install is required_

* `git clone https://github.com/apollo-agit/cms-upload-poc.git`
* `cd cms-upload-poc`
* `npm install -dd`
* `bower install`
* `grunt dist`
* `grunt server watch`
* open web browser and navigate to http://localhost:8888/
